# Godot Cursor Template

A ready-to-use **Godot 4.6** project template designed for development with [Cursor](https://cursor.com) AI agents. Clone this to start a new game with CI/CD, Web export, and GitHub Pages deployment already configured.

## What's included

- **Godot 4.6** project configured for GL Compatibility (desktop + mobile + web)
- **Web export** preset (`export_presets.cfg`) targeting `Builds/`
- **GitHub Actions** workflow that builds the Web export and deploys to GitHub Pages on every push to `main`
- **GDAI MCP plugin** for Cursor agent integration
- **AGENTS.md** with instructions for Cursor Cloud agents
- Minimal placeholder scene (`scenes/main.tscn`) and script (`scripts/main.gd`) so the project opens without errors

## Getting started

1. Click **"Use this template"** on GitHub (or clone directly)
2. Open the project in Godot 4.6.1
3. Rename the project: edit `config/name` in `project.godot`
4. Replace `scenes/main.tscn` and `scripts/main.gd` with your game content
5. Enable GitHub Pages: repo **Settings → Pages → Source: GitHub Actions**
6. Push to `main` — your game will be live at `https://<you>.github.io/<repo>/`

## Project structure

```
├── scenes/          # Game scenes (.tscn)
├── scripts/         # GDScript source files (.gd)
├── addons/          # Godot addons (includes GDAI MCP plugin)
├── Builds/          # Web export output (git-ignored, generated by CI)
├── export_presets.cfg
├── project.godot
├── AGENTS.md        # Cursor agent instructions
└── .github/workflows/godot-web-pages.yml
```

## Requirements

- Godot Engine **4.6.1**
- For web export: export templates installed at `~/.local/share/godot/export_templates/4.6.1.stable/`
